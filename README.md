uCampus++ 

uCampus++ is a smart campus portal designed to centralize student activities, updates, and services in one interactive platform. The platform enhances communication, improves transparency, and helps students stay informed about campus life efficiently.

The project focuses on creating a clean, responsive, and user-friendly experience that helps students access important campus resources efficiently.

 Features:
📢 Campus Event Listings
💻 Hackathon Updates
🏫 Club Announcements
🚦 Student Traffic Tracking System
🍱 Girls’ Hostel Mess Menu
💰 Food Price Comparison System
⭐ Student Reviews & Feedback
🔐 Login Authentication System
🌙 Dark Mode Support
🎞️ Animated Slideshow Banners
📊 Animated Statistics Dashboard
📱 Fully Responsive Modern UI
🛠️ Admin Panel for Dynamic Event Management
